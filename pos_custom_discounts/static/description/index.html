POS Custom Discount

Provide custom discounts for satisfied customers!!

POS Custom Discount allows sellers to give discounts on a single order line as well as on the whole order. When you click on the discount button a new window will appear which contains those discounts that are configured from the backend. It also allows seller to create custom discounts(new discounts) in running POS sessions.

Custom Discounts: A perfect way to lure customers in POS!!

Whenever we hear discounts, we all become ready to look up the products and start shopping. Discount is the first thing for which a customer reaches out before buying a product. Even if there is a single rupee discount, it will appeal to customers and will compel them to go for instant purchase.

Providing discounts on the order line makes customers happy and satisfied and gives you the guarantee of future purchases too, i.e you are efficient to retain reliable and valuable customers. With changing the old functionality of the discount button, the module provides discounts on a single order line as well as on the whole order.

On selecting discount from available discounts
  • After selecting discount from discount list, if seller click on selected line button then discount will be applied on selected product/item in cart.
  • After selecting discount from discount list, if seller click on whole order button then discount will be applied on all products/items in cart(complete order).
    • Creating custom discount
  • Click on custom discount button to create a custom discount
  • After clicking on custom discount button, user authentication will be done. A valid POS user needs to enter Security Pin to create custom discount. This ensures that only the authorized POS user can create custom discount.
  • After password authentication,this window will be shown as shown below. It contains discount percentage and discount reason. Reason for custom discount will be mentioned on receipt, orderline and get saved in backend also.
  • If seller click on selected line button then discount will be applied on selected order line(selected product in cart) with reason.
