Employee Time Off Portal (Employee Leave Portal)

- This module is to allow the Portal users can manage Time Off request from Portal User Account.

Installation

- Copy employee_timeoff_portal module to addons folder

- Install the module normally like other modules

Set portal access to your user

Settings --> Users&Companies --> Users --> User Type --> Portal



Employees --> Related User --> Portal User
